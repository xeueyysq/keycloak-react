## Обзор

Это демонстрационное приложение, показывающее интеграцию React с Keycloak для аутентификации и авторизации. Приложение включает:

- Аутентификацию через Keycloak
- Ролевой доступ
- Адаптивный дизайн
- Поддержку PWA
- Темную/светлую тему
